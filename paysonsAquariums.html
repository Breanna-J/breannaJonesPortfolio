VIDEO DEMOInventory Webapp

While working through this WebApp, I was able to strengthen my Java, HTML, and SQL skills while gaining a strong foundation in Spring Boot, Bootstrap, Hibernate and Thymeleaf.
This application allows the user to add, update, or delete a product, along with its associated parts, to or from an inventory repository. The user is also able to assign minimum and maximum part inventory quantities, which will prevent the adding and removing of the product or part if these thresholds are passed.
